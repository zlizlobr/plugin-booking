# Booking Form Block

Block Name: `booking/booking-form`  
Block Icon: `email-alt`

## Description
Display a booking form with date picker input field with calendar options

## Configuration Options

### Main Settings

#### booking_id
- **Type:** post
- **Label:** Booking
- **Description:** Select booking
- **Post Type:** booking
- **Position:** inspector

#### open_in_new_window
- **Type:** toggle
- **Label:** Open form in new window
- **Description:** When enabled, form will save data and open the target URL in a new window instead of advancing to the next step. Only step 1 will be displayed.
- **Default:** false
- **Position:** inspector

## Usage
This block allows you to embed a booking form into any page or post. Select the booking you want to display and configure whether it should open in a new window.
